# PodTidy — 播客音频整理工具

一键将拖入的播客音频文件自动批处理：识别播客归属、格式化 ID3 标签、标准化文件名、调整音频增益，最终自动分发到对应的播客文件夹。

## 使用方法

### 直接运行（打包版）

1. 从 [Releases](../../releases) 下载 `PodTidy.zip`
2. 解压到任意目录
3. 双击 `PodTidy.exe` 启动

### 从源码运行

```bash
pip install -r requirements.txt
python main_gui.py
```

## 操作流程

1. **选择播客目录** — 点击「播客目录...」选择一个根目录，目录下应有以播客名命名的子文件夹（如「得体广播站」「三个火呛手」等），每个文件夹内需放置 `folder.jpg` 或 `folder.png` 作为专辑封面
2. **拖入音频文件** — 将 MP3/M4A/FLAC 等音频文件拖入虚线区域（或点击浏览）
3. **点击「开始整理」** — 软件自动完成以下处理

## 处理内容

| 步骤 | 说明 |
|------|------|
| 播客识别 | ID3 标签优先（artist/album），文件名关键词回退 |
| 标签格式化 | 清除旧标签，写入 artist / album / title / tracknumber |
| 专辑封面 | 清除原封面，嵌入对应播客文件夹中的 folder.jpg |
| 文件名格式化 | 统一为 `{播客名} - {音轨号} - {标题}.mp3` |
| 音频增益 | ReplayGain 音轨增益扫描（等效 foobar2000），已有增益则跳过 |
| 自动分发 | 复制到对应播客子文件夹，**原文件保留不动** |

> **提示**: 已用本软件整理过的文件重新拖入时，标题和音轨号会直接从 ID3 标签读取，无需依赖文件名。支持对已有文件进行批量重命名。
